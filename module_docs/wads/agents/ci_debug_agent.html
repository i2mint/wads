wads.agents.ci_debug_agent

CI Debugging Agent for Wads

This agent analyzes failed CI runs, diagnoses issues, and proposes fixes. It can: - Fetch GitHub Actions logs via API - Parse test failures and error messages - Analyze code context - Diagnose root causes (missing deps, config issues, code bugs) - Propose fixes (install commands, code changes, config updates)

Usage:

python -m wads.ci_debug_agent <repo> [–run-id RUN_ID] [–fix]
