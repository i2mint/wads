wads.agents

Wads AI Agents

Autonomous agents for diagnosing and fixing common development issues.

Available Agents: - ci_debug_agent: Analyzes failed CI runs and proposes fixes - dependency_resolver: Analyzes import errors and missing dependencies - test_analyzer: Analyzes pytest failures and categorizes them
