wads.scrap.handle_non_python_installs

Handle non python installs, depending on the system

wads.scrap.handle_non_python_installs.make_install_instructions_for_ci(commands_for_os: dict) → str[source]
Parameters:

commands_for_os – A dictionary containing the commands for each OS,

where the keys are the OS names (as in os.name) and the values are the commands, in the form of a list of strings that can be passed to os.system to be executed. :return: A string that can included in a github actions ci.yml file, which will contain the os specific commands to install the package.

>>> make_install_instructions_for_ci({'Linux': ['pip install .']})
'- name: Install on Linux\n  run: |\n    pip install .\n'
wads.scrap.handle_non_python_installs.make_install_instructions_for_readme(commands_for_os: dict) → str[source]
Parameters:

commands_for_os – A dictionary containing the commands for each OS,

where the keys are the OS names (as in os.name) and the values are the commands, in the form of a string that can be passed to os.system to be executed. :return: A string that can included in the “How to install” section of a README.md, which will contain the instructions for the user to install the package, listing the commands for each OS.

>>> make_install_instructions_for_readme({'Linux': 'pip install .'})
'### Linux\n\n```\npip install .\n```\n'
